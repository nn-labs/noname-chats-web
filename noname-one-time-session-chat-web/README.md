# Get started

### 1. You need clone and run backend [repo](https://github.com/nn-labs/noname-realtime-support-chat)

### 2. Setup envs
```
REACT_APP_API_URL=
REACT_APP_WS_URL=

REACT_APP_ALGORITHM=
REACT_APP_ITERATION=
REACT_APP_KEY_LEN=
REACT_APP_DIGEST=
REACT_APP_CRYPTO_SECRET=
```

### 3. Run app
```npm start```
